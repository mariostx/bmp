# Build and run using docker
Go to the BMP root folder and run commands from the CLI.

<br/>

## Build and run
```
docker-compose build
docker-compose up -d
```

<br/>

## List containers
```
docker-compose ps
```

<br/>

## Stop containers
```
docker-compose down
```

<br/><br/>
